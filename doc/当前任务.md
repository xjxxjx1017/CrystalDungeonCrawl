*预定任务*

第二版的游戏 - 类魔塔：
策划目标
1. 伤敌一千，自损八百! 每一步的思考，每一份资源的精打细算。鲁莽的选择会导致损失。
2. 无限! 无限刷的关卡。值得刷。虽然在刷，但是每次体验不一样。虽然在刷，但却没有通货膨胀。
3. 选择自由! 从初始技能到终极OP技能，玩家一开始就有所有的选择自由，但是不一定付得起代价。
4. 计分制！没有最高分，只有更高分。无上限的最高分挑战，让玩家由不断进步的可能。

策划原则分析
1. 玩家技术阶段分析. 
   1. 低等级的玩家，无法使用超强的技能，并且通关得分低，而且需要尽量打完所有的怪。
   2. 高等级玩家可以快速通关，更偏向于使用强力技能，通关得分高。
2. 怪物与玩家技能的相互克制
3. 玩家得流派分析
4. 关卡设计最好能卡死一些, 设计waypoint，用于方便策划计算和设计难度阶梯。
   1. 当前得waypoint是officer和boss
5. 玩家技巧分析
   1. 在适当的面板情况使用适当的招式组合
   2. ? 如何给玩家尽量多的选择 ?
   3. 是先拿剑，还是先拿盾？玩家选择对于结果的影响
   4. 是解锁当前技能，还是升级当前技能 ？
6. 解锁系统分析
   1. 一开始需要暴露所有系统
   2. 熟练玩家可以快速解锁多次多个
   3. 总是提供玩家还没尝试的路径

*思考一下gameplay的优化，怎么让数值更紧凑，游戏更难。还需要在大地图上做文章。*
1. 允许武器有过强或者OP的技能。
2. 技能使用得越多，等级越高？等级高则威力大，但是消耗会增加，虽然会打折扣. (每个技能有自己的等级，越是使用，等级越高)。技能等级越高，越是考验玩家对于游戏得理解。
3. 玩家走路逐渐恢复HP，踩在火上恢复HP增加。
4. 感觉还是需要更明确的在地图上的目标，每一个目标会产生压迫感，与疏解感。有的地图目标会有惊喜感。有的地图目标会让玩家做思考与选择。
5. 让玩家自己解锁，选择技能。如果玩家使用同一流派技能，则会有(经验值/威力/分数)加成？或者解锁其它技能树，需要更多得资源
6. 特效技能可以给技能加上前缀后缀。

更少量的剑与盾 - 剑盾的位置都距离玩家跟BOSS稍微远，并且相互之间有距离。

可以把思考的点向售前倾斜，从玩家看到图标和标题开始，去想会不会点进来，点了会不会买
